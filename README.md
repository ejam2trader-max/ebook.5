# Rut y Boaz - eBook Pro por GracePrint

Este es un proyecto de eBook ilustrado que narra la historia bíblica de Rut y Boaz, enfocándose en la lealtad y la gracia de Dios.

## Características
- 10 Ilustraciones en alta definición (WebP).
- Diseño responsivo para lectura en móviles y tablets.
- Narrativa adaptada para niños.
